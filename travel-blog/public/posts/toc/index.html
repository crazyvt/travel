How to Add New Destinations

To add a new destination to this index:

  1. Create the directory structure: content/posts/country/region/place/
  2. Add the destination’s index.md file with proper front matter
  3. Update this TOC file with the new entry
  4. The new destination will automatically appear in the site navigation
